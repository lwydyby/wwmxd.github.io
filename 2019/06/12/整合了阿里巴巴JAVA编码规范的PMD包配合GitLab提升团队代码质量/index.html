docker 补充(其他见简书上的那篇博客)：
由于我这边的gitlab是通过docker搭建的，所以我这边还需要考虑如何在docker中安装java
具体方式如下：
1.在宿主机上下载并解压jdk，如jdk8
2.通过volume来共享给gitlab容器
3.（可选）如果你那边有管理docker容器的应用（如rancher），可以添加环境变量，这样可以直接使用java -version等命令 （没有的话可以直接用绝对路径运行，如：/usr/local/jdk/bin/java -cp …）

注：原作者的脚本有个坑，如果按照他那个脚本，多文件必定报错，因为-d 后应该传的的是临时的文件夹而不是多个要检查的文件 ，files_to_check 这一步可以不需要，应该传的是TEMPDIR
